传播途径：经呼吸道飞沫传播，亦可通过接触传播，存在粪-口传播可能性

易感人群：人群普遍易感。老年人及有基础疾病者感染后病情较重，儿童及婴幼儿也有发病

宿主：野生动物，可能为中华菊头蝠
